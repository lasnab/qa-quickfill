Disclaimer

This extension is designed to store generic login information for test accounts and is not intended to be a password manager. While efforts have been made to ensure the security of stored data, there are no guarantees of complete security. It functions more like a diary, allowing you to write down and autofill certain fields in a form for testing purposes, as well as keeping track of relevant information. Please use this extension responsibly and avoid storing sensitive or personal login credentials.
